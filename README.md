# SpaceX

Небольшой учебный проект — лендинг, вдохновлённый сайтом SpaceX.  
Сделан для практики адаптивной верстки и работы с Bootstrap.

## Демо

https://sxnizh.github.io/spacex-landing/

## Технологии

- HTML
- CSS
- Bootstrap 5

## Что есть в проекте

- адаптивная навигация
- сетка Bootstrap
- адаптивные изображения
- футер с ссылками и кнопкой

## Как запустить

1. Клонировать репозиторий
git clone [https://github.com/sxnizh/spacex-landing.git](https://github.com/sxnizh/spacex-landing.git)

2. Открыть папку проекта
cd spacex-landing

3. Открыть `index.html` в браузере
